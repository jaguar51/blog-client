# Blog-client

The client part of the blogging system implemented RectJS.   
Server part you can see [here](https://github.com/jaguar51/Blog-server).

## Install
1. Clone the project
2. Build project

	```bash
	npm run build
	```
3. Place the application on your server

## Run for develop
1. Clone the project
2. Run server mode

	```bash
	npm run server
	```
3. Development
